Edits here wait until you publish. The Studio Dock counts them and tells you what’s left.

One coherent system, not a skin. Every color is authored in OKLCH and every state derives from a single hue token via color-mix() — there is zero literal hex in the component CSS, so a buyer re-hues the whole template by editing the hue tokens. No webfonts, no backdrop-filter, no build step. Dark mode is deferred but not foreclosed: the lightness channel is kept clean for a future prefers-color-scheme remap.
