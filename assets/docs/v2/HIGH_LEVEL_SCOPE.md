## Project Details 

### Technical Infrastructure

**Custom-built dynamic website:**
- JSON-based product catalog system
- GitHub Pages hosting (free, fast, reliable)
- Cloudflare R2 CDN for images (~$1-5/month)
- Automated deployment pipeline
- Future-proof, AI-agent-ready architecture
- Vercel serverless functions 

**E-commerce integration:**
- Stripe Prebuilt Checkout (industry standard)
- All payment methods (cards, Apple Pay, Google Pay, etc)
- Automatic inventory updates
- Webhook integration for sold items
- Mobile-optimized checkout experience

**Operating costs:**
- Domain: $12/year (registered through Feb 2027 on Squarespace)
- Google Workspace: $72/year/seat (already active)
- Image CDN: ~$12-60/year
- Stripe fees: 2.9% + $0.30 per transaction (standard)
- **Total: ~$96/year** 

### Design & User Experience

**Brand-aligned visual design:**
- Your color palette (plum, lavender, fog, cream, gold)
- Minimalist, elegant aesthetic
- Theatrical lighting effects
- Mobile-first responsive design
- Professional typography

**Strategic UX features:**
- Single landing page controlling narrative flow
- Price above fold (conversion best practice)
- Buy Now buttons next to add to cart buttons
- Prominent newsletter CTA in landing page and all footers 
- Product catalog grid view with multi-select tag filters
- Professional template-based product pages

#### Website Features

**Homepage:**
- Theatrical opening, loading like a curtain opening on a stage or a book opening to show a story 
- Perspective-motion UI lighting effects mimic stage spotlights crossing scene before curtain open
- Seasonal theme system that dynamically adjusts aesthetics pulling from selected featured product JSON files 
- Brand story introduction using landing page structure best practices, including Newsletter, Contact, and Shop CTAs
- Homepage contact expands into modal with contact form and defaulted newsletter sign-up

**All Pages:**
- Header, footer, and breadcrumbs making for easy navigation between pages and page sections 
- Mobile first design to accommodate social ad traffic purchase behavior 
- Hamburger menu for detailed navigation leaving room for 3-5 primary navigation links 

**Collection Pages:**
- Section-specific hero images that pull from the project JSON files for featured products 
- Filterable product grid with multi-select tags, sort controls, smart filtering hiding irrelevant filters
- Product cards with hover effects, quick view, add to cart, and buy now buttons 

**Product Pages:**
- Photo galleries with 7-15 images
- Story cards (2-8 paragraphs)
- Feature lists with icons, full product specifications
- Related products connected using tagging system

**External Features:** 
- Google Analytics 4 setup with custom events that make sense for the business
- Pixel tracking for Meta, TikTok, and Pinterest
- Social media integration, image alt tags, social meta tags
- SEO optimization, potential innovative AI-driven solution; product description updates 

### Content Management

**Unlimited scalability:**
- Add new products by creating JSON files; script to auto-generate prepped JSON file for new product entry 
- New sections appear automatically thanks to dynamic page generation using sections as type of tag 
- Tag system adapts to your creations; no rebuilding pages or touching code
